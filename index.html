Telangana Diagnostics


This application is an extension to Telangana Diagnostics Web Portal which provides free patient diagnostic services across the state of Telangana using Hub & Spoke Model since 2018.


In order to make the Spoke / Sample Collection operations easier and faster, this new Mobile application had been rolled out which enables the end user to do Patient Registration, Test Assignment, Barcoding and Consignment Preparation.

Drugs control Administration


The Drugs Control Administration was extended to Telangana region after the formation of Andhra Pradesh State in the year 1956 when four Drugs Inspectors were appointed and implementation of Drugs & Cosmetics Act was extended to this area.

This app is exclusively intended for department employees.
